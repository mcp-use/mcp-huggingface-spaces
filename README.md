# Hugging Face Spaces — MCP App

Search and embed Hugging Face Spaces directly in your chat. Browse trending AI demos, search by keyword, and open any Space as a live iframe.

Built with [`mcp-use`](https://mcp-use.com).

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000/inspector](http://localhost:3000/inspector) to test your server.

## Tools

- **search-spaces** — Search HF Spaces by keyword, returns a browsable grid
- **show-space** — Embed a specific Space by its `author/name` ID
- **trending-spaces** — Browse what's trending on Hugging Face

## Deploy

```bash
npm run deploy
```
